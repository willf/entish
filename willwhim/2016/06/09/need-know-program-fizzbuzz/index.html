What do you need to know to program FizzBuzz?

By willwhim

June 9, 2016

I was showing my CS professor friend the hilarious “FizzBuzz in TensorFlow” post, and he’d actually had not heard of FizzBuzz. Basically FizzBuzz is a test to see if someone can do basic programming.

This got me to thinking about what you actually need to know about a particular programming language and software engineering to program FizzBuzz in.

So I think these are the things you need to know (including the syntax of these in a particular language):

  • How to print something
  • How to write a for loop
  • How to write a select statement (or if/then/else cascade)
