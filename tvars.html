Possible forms for indexical variables

Indexical variables are really function calls; when the variable 'evaluator' sees an indexical variable, it calls the function by the same name (with a task as its parameter). Remember that currently ?var refers to a 'standard' variable, and that <?var> refers to a variable that should be bound at its time of access. The question is: how should indexical variables be notated? I think this list contains the total set of reasonable possibilities. Which do you think is clearest? this-task is the current task; now is the current time; self is the agent of the current task; tot is time on task; met is mission elapsed time (time since application start).
