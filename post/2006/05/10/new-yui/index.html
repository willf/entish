New YUI

By willwhim

May 10, 2006

Yahoo! has announced a new rev of the Yahoo! User Interface library. I’ve already thought of an innovative use for the windowing controls, which must remain sekrit for now. Safari died a spectacular death when testing out the AutoComplete feature, though. I guess that’s one reason they’re still calling it a beta test. I really wish I’d had this library available when I did the NASA mission planning interface.
