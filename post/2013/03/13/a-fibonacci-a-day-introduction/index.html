A Fibonacci A Day Introduction

By willwhim

March 14, 2013

The Fibonacci sequence is the sequence of numbers 0,1,1,2,3,5,8, … such that the nth number in the sequence is the sum of the previous two numbers; and the the first two numbers are defined to be 0 and 1.

The Fibonacci function is a function which returns the nth number in the Fibonacci sequence. It’s kind of a famous function in computer science when people are first learning about recursive functions. It’s sometimes used as a kind of FizzBuzz test — a programmer ought to be able to write the Fibonacci function quickly. But there are lots of things that are going on even with something as simple as the Fibonacci function, and I’d like to explore these for a while in a series that I’ll call “A Fibonacci A Day” (with a nod to Erin McKean’s A Dress A Day, only with more programming and less stylistic flair).

All the programming will be in Scala, and code will be in a GitHub repository, fibonacci-a-day. Tomorrow, we’ll start with perhaps the most famous version, the naive recursive algorithm.
